## 二、每日发布流程（5 步）

每天凌晨 00:20，`run_daily` 自动生成当日物料到 `backend/ops/out/YYYY-MM-DD/`。

抖音文案和 H5 首页轮播**来自同一个选题**，用户从抖音进来能看到对应的长文。

```
1. 打开 backend/ops/out/今天日期/douyin_publish.md
2. 复制「标题」代码块 → 粘贴到抖音标题
3. 复制「正文」代码块 → 粘贴到抖音描述
4. 复制「话题标签」代码块 → 粘贴到抖音描述末尾
5. 上传 media/images/ 下的图片（page_01.png 等）
6. 发布后，复制「置顶评论」代码块 → 粘贴为评论并置顶
```

### 目录文件说明

| 文件 | 用途 |
|------|------|
| `douyin_publish.md` | **核心**：标题+正文+标签+配图路径+置顶评论，全部代码块可复制 |
| `media/images/page_*.png` | 配图，直接上传抖音 |
| `pinned_comment.txt` | 置顶评论全文 |
| `media/traffic_qr.png` | 引流二维码（如有） |
| `copy.md` | 完整运营稿（长版，备参考） |
| `voice.txt` | 口播提纲（录口播视频用） |
| `douyin_compliance.txt` | 合规检查（发前看一眼） |
| `manifest.json` | 元数据（不需要操作） |

### 手动生成（如果凌晨没自动跑）

```bash
cd backend && .venv/bin/python -m ops.cli daily
```

## 七、内容对齐说明

抖音短文案和 H5 首页轮播由**同一条流水线**（`ops/pipeline.run_daily`）生成：

- 同一个选题引擎（`CandidateAngle` + `rank_angles`）决定今天推什么
- 抖音：短文案 + 配图（`ops/out/` 下，人工复制发布）
- 轮播：长文版本（自动写入 DB `articles` 表，H5 首页自动展示）

用户从抖音看到"狮子座今天xxx"，点进 H5 首页，轮播第一张就是对应的狮子座长文。
